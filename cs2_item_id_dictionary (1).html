cs2_base_info 缓存数据说明

数据来源

本页展示的数据来源于 SteamDT 开放平台，当前网页会读取已经缓存好的 CS2 物品基础信息。

缓存地址

https://steamauto.jiajiaxd.com/file/cache?key=cs2_base_info

顶层 JSON 结构

{
  "success": true,
  "data": []
}
字段类型说明
successboolean请求或缓存读取是否成功。
dataarray物品基础信息数组，每一项代表一个 CS2 物品。

data 数组元素结构

{
  "name": "AWP | 响尾蛇 (略有磨损)",
  "marketHashName": "AWP | Pit Viper (Minimal Wear)",
  "platformList": [
    { "name": "BUFF", "itemId": "34123" },
    { "name": "C5", "itemId": "22313" },
    { "name": "YOUPIN", "itemId": "2923" },
    { "name": "HALOSKINS", "itemId": "22313" }
  ]
}
字段类型说明
namestring物品中文名。
marketHashNamestringSteam 市场 Hashname，可用于和 Steam / BUFF 等市场数据匹配。
platformListarray各平台的物品 ID 映射列表。
platformList[].namestring平台名称，例如 BUFF、C5、YOUPIN、HALOSKINS。
platformList[].itemIdstring该物品在对应平台上的 ID。

Python 使用示例

import requests

CACHE_URL = "https://steamauto.jiajiaxd.com/file/cache?key=cs2_base_info"

resp = requests.get(CACHE_URL, timeout=20)
resp.raise_for_status()
payload = resp.json()

items = payload["data"]

# 建立两个常用索引：按 Hashname 查、按 BUFF ID 查
by_hashname = {}
by_buff_id = {}

for item in items:
    name = item.get("name", "")
    hashname = item.get("marketHashName", "")
    platforms = item.get("platformList", [])

    platform_ids = {
        p.get("name"): p.get("itemId")
        for p in platforms
        if p.get("name") and p.get("itemId")
    }

    row = {
        "name": name,
        "marketHashName": hashname,
        "platformIds": platform_ids,
    }

    if hashname:
        by_hashname[hashname] = row

    buff_id = platform_ids.get("BUFF")
    if buff_id:
        by_buff_id[buff_id] = row

# 示例查询
print(by_hashname.get("AWP | Pit Viper (Minimal Wear)"))
print(by_buff_id.get("34123"))
